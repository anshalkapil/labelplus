LabelPlus allows you to Label data sets to prepare them for Supervised Machine Learning or any other use case.

Step to use LabelPlus:

1. Convert your dataset into CSV
2. Upload it to the program
3. Select number of distinct Labels you want to classify the data points into.
4. Assign the labels for the available keyboard shortcuts.
5. Enter the index (Row number starting from 0) from where you want to begin labelling
6. Keep pressing the appropriate keyboard shortcut to label your dataset quickly.
7. Press Finish once you're done.
